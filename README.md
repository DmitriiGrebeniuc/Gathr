Gathr is a mobile-first web app for discovering, creating, joining, and sharing local events.

Current product areas in the repository:
- authentication with login, signup, logout, forgot-password, and recovery reset flow
- event discovery, creation, editing, joining, leaving, and deletion
- participants list and creator-driven invitations
- notifications for upcoming events, new participants, and event invitations
- profile, language, security, support, and notification settings
- realtime UI updates through Supabase
- Android packaging through Capacitor
